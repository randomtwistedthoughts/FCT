# Practice Questions

Answers

1. Problem 1 — What is a list?
   - Answer:
     A list is an ordered collection of values.

2. Problem 2 — How do you access list values?
   - Answer:
     Using square brackets with an index.

3. Problem 3 — What is the index of the first element in a list?
   - Answer:
     0

4. Problem 4 — What happens when you use a negative index?
   - Answer:
     It accesses elements from the end of the list.

5. Problem 5 — What does `len()` return for lists?
   - Answer:
     The number of items in the list.

6. Problem 6 — Are lists mutable?
   - Answer:
     Yes, lists can be changed.

7. Problem 7 — What does `list[2:5]` return?
   - Answer:
     A slice containing elements at index 2, 3, and 4.

8. Problem 8 — What does `in` check for in a list?
   - Answer:
     Whether a value exists in the list.

9. Problem 9 — How do you add a value to the end of a list?
   - Answer:
     Using `append()`.

10. Problem 10 — How do you combine two lists?
    - Answer:
      Using `+` (concatenation).

11. Problem 11 — What does `remove()` do?
    - Answer:
      Removes the first matching value.

12. Problem 12 — What is the difference between `sort()` and `sorted()`?
    - Answer:
      `sort()` sorts in place; `sorted()` returns a new list.

13. Problem 13 — How do you loop over a list?

        for item in my_list:
            print(item)

14. Problem 14 — What is the difference between a list and a tuple?
    - Answer:
      Lists are mutable; tuples are immutable.

15. Problem 15 — What does `my_list.index('a')` do?
    - Answer:
      Returns the index of the first occurrence of `'a'`.

16. Problem 16 — How do you insert a value at index 2?
    - Answer:
      `list.insert(2, value)`
