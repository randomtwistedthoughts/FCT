# Practice Questions

Answers

1. Problem 1 — What is a function?
   - Answer:
     A function is a named sequence of statements that performs an action.
   - Explanation:
     Functions allow you to reuse code and avoid repetition.

2. Problem 2 — What is the difference between a function and a function call?
   - Answer:
     A function is a block of code; a function call executes that block.
   - Explanation:
     Calling the function runs the statements inside it.

3. Problem 3 — Where does execution start when Python runs a program?
   - Answer:
     The first line of the program, not inside function definitions.

4. Problem 4 — What is the difference between `return` and `print()`?
   - Answer:
     `return` sends a value back to the caller; `print()` shows text on screen.

5. Problem 5 — What does the following code output?

        def spam():
            return 'Hello'
        print(spam())

   - Answer: `Hello`

6. Problem 6 — What is `None`?
   - Answer:
     A special value representing “no value”.

7. Problem 7 — What is the difference between local and global scope?
   - Answer:
     Local variables exist inside a function; global variables exist outside.

8. Problem 8 — What happens if you try to access a local variable outside its function?
   - Answer:
     You get a `NameError`.

9. Problem 9 — How do you indicate parameters in a function?
   - Answer:
     By listing them in parentheses after the function name.

10. Problem 10 — What is the purpose of a docstring?
    - Answer:
      To describe what a function does.

11. Problem 11 — Write a function that adds two numbers and returns the result.

        def add(a, b):
            return a + b

12. Problem 12 — What is a keyword argument?
    - Answer:
      An argument passed by name instead of position.

13. Problem 13 — What statement creates an infinite loop?
    - Answer:
      `while True:`

14. Problem 14 — What does the `global` statement do?
    - Answer:
      It lets you modify a global variable inside a function.
